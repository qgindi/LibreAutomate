# Version 1.16.0 (2026-07-06)

## Editor

Using .NET 10.

Improved in **NuGet** tool:
- Restores old folder contents if failed to install/update.
- Can update dlls that are currently loaded in script processes.

New in **Help** panel:
- Favorites. Right-click...
- In web browser opens local pages by default.

Fixed bugs:
- Was defined `NET8_0`.
- Some types missing in intellisense.
- The drag-drop-file menu disappears.

Removed features:
- Completion list filters, like `global using Namespace; //[filters]`.

## Library

Fixed bugs:
- `csvTable` incorrectly parses CSV that ends with comma without new line.
